## Requisitos

Tener instalado nodejs y npm en tu máquina.

##Carga el entorno

Clona este repositorio y luego ejecuta dentro de su carpeta lo siguiente:

```
npm install -g grunt-cli
npm install

```

##Ejecución (para desarrollo)

```
grunt
```

##Compliación (para despliegue)

```
grunt build
```
